# Widget_UI
常用UI控件集合

[![](https://www.jitpack.io/v/com.gitee.kframe/widget_ui.svg)](https://www.jitpack.io/#com.gitee.kframe/widget_ui)

#### 使用方式

##### 1. 将JitPack库添加到全局grade文件中

~~~
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
~~~

##### 2. 添加依赖项
~~~
dependencies {
    implementation 'com.gitee.kframe:widget_ui:XXX'
}
~~~

> 如果有引用包冲突和不必要的包，请排除
